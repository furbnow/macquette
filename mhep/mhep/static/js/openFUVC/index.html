Open floor U-value calculator (openFUVC)

Welcome to openFUVC, an open source implementation of the thermal transmittance calculation of floors as specified in BS EN ISO 13370:2007.

This project has been developed by URBED and Carbon Co-op as part of MyHomeEnergyPlanner.

The calculator is released under the GNU Affero General Public License. So download it, study it, share it, change it, use it for yourself or in your website in which case don't forget to add a link to the source code (also applies to any changes you may make).
